Plugin custom per la gestione completa dell'Informagiovani del Comune di Enna.

== Description ==

Plugin WordPress custom (non React, non tema) che implementa:

* Schede informative (bandi, opportunità, concorsi)
* Eventi
* Ticket / richieste utenti
* Appuntamenti e colloqui di orientamento
* Partner ed enti collegati
* Percorsi impresa
* Area personale utente
* Backend gestionale per operatori
* Report, newsletter, notifiche, export
